Instructions to setup PyBlock
  1. PyBlock is a tool for viewing and interacting with your node.
  2. To get started, you need access to the Linux terminal of your myNode device.
    • A guide to access the Linux terminal is available on mynodebtc.com - guide
  3. Once you have terminal access, run the following command:
    • sudo mynode-pyblock
  4. Enter your myNode password when prompted.
  5. The first time running PyBlock, you may be prompted for various node information.
    • For more information on filling it out, see the PyBlock GitHub page.
  6. Enjoy PyBlock!
